#A friendly smart calculator

A program which allows the user to pick which type of an operation they want to carry out in their calculation (subtraction,addition,multiplication,division).
Then gives out an output based on it's calculation.At the end,it also asks the user if they would like to continue using the program,which allows the user to do more calculations or exit the program.

What I learnt:
-while loops

-defining functions

-conditional statements(if,else)

-dictionaries(operations)

-lists(different types of inputs, which result in the same function).
